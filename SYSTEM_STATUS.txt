🚀 WHAT NEEDS TO BE DONE (ONLY 3 SIMPLE STEPS!)
═══════════════════════════════════════════════════════════════════════════════

Step 1: Install Redis (Choose 1 - Takes 5 minutes)
  □ Docker (Easiest):
    docker run -d -p 6379:6379 redis:latest

  □ WSL Ubuntu:
    wsl --install -d Ubuntu
    (Then in WSL): sudo apt install redis-server

  □ Online (No setup):
    Go to: redis.com/try-free/

  □ Windows (Native):
    Download from: github.com/microsoftarchive/redis

Step 2: Start 4 Services (Open 4 terminals)
  Terminal 1: redis-server (or docker start redis)
  Terminal 2: npm run dev
  Terminal 3: node src/services/worker/MessageWorker.js
  Terminal 4: cd client && npm run dev

Step 3: Open Browser
  http://localhost:3000
