┌─────────────────────────────────────────────────────────────────────────────┐
│ ⚠️  TROUBLESHOOTING QUICK FIXES                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Problem: "Cannot find module"
  Fix: npm install && cd client && npm install

Problem: "Port already in use"
  Fix: netstat -ano | findstr :5000  (find process)
       taskkill /PID xxxx /F         (kill process)

Problem: "Redis connection failed"
  Fix: docker run -d -p 6379:6379 redis:latest
       OR: redis-server (if installed locally)
       OR: redis-cli ping (verify connection)

Problem: "MongoDB connection failed"
  Fix: node test-db-connection.js (check connection)
       Verify .env has correct MONGODB_URI

Problem: "Blank page at http://localhost:3000"
  Fix: Ctrl+Shift+R (hard refresh, not F5)
       Check browser console (F12) for errors
       Verify Terminal 4 shows "Local: http://localhost:3000"

Problem: "Socket.IO connection failed"
  Fix: Verify Terminal 2 shows "Server running on port 5000"
       Check browser console for connection errors
       Restart Terminal 2 (npm run dev)

Problem: "WhatsApp QR code not showing"
  Fix: Refresh page
       QR expires in 60 seconds - scan quickly
       Check Terminal 2 logs for WhatsApp errors
